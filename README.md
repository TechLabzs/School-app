# School Assistant App

A fun and interactive app designed for kids at school to help them manage lessons, communicate with classmates, do math homework, and keep track of time!

## 🎓 Features

### 📚 **Lesson Reminders**
- Add reminders for upcoming lessons
- Set specific times for each lesson
- Choose from various subjects: Math, English, Science, History, PE, Art, Music, and IT
- View all reminders in beautiful card format
- Delete reminders you no longer need
- Get notifications when it's time for a lesson

### 💬 **Class Chat**
- Send and receive messages in real-time
- Simulated responses for fun interaction
- Message history is saved locally
- Easy-to-use chat interface

### 🧮 **Calculator**
- Full-featured calculator for math homework
- Basic operations: Addition, Subtraction, Multiplication, Division
- Decimal support
- Clear button to reset calculations
- Beautiful, colorful interface

### 🕐 **Clock & Time**
- Digital clock showing current time and date
- Analog clock with moving hands
- Real-time updates every second
- Helps kids stay on schedule

## 🚀 Installation & Usage

### Method 1: Direct File Usage
1. Download all three files: `index.html`, `style.css`, and `app.js`
2. Keep them in the same folder
3. Open `index.html` in your web browser
4. Start using the app!

### Method 2: On Your Legion Go
1. Download the repository as ZIP from GitHub
2. Extract the files on your device
3. Open `index.html` in your browser
4. Works perfectly on handheld devices!

### Method 3: Web Server
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx http-server
```
Then open `http://localhost:8000` in your browser.

## 🎨 How to Use

### Adding a Lesson Reminder
1. Go to the **Reminders** tab
2. Enter the lesson name (e.g., "Math Test")
3. Select the time using the time picker
4. Choose the subject from the dropdown
5. Click "Add Reminder"
6. Your reminder will appear as a card below

### Chatting
1. Click the **Chat** tab
2. Type your message in the input field
3. Press Enter or click "Send"
4. The app will respond with a friendly message!

### Using the Calculator
1. Go to the **Calculator** tab
2. Click number buttons to enter digits
3. Click an operator (+, −, ×, ÷)
4. Enter the next number
5. Click "=" to get the result
6. Click "Clear" to start over

### Checking the Time
1. Click the **Clock** tab
2. View the digital time and date
3. Watch the analog clock hands move in real-time

## 🔧 Customization

### Change Colors
Edit the color values in `style.css`:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Add More Subjects
In `index.html`, add more options to the select element:
```html
<option value="🚀 Computer Science">🚀 Computer Science</option>
```

## 🐛 Troubleshooting

### Messages not sending?
- Make sure you've typed something in the message box
- Try refreshing the page

### Reminders not appearing?
- Make sure to fill in all three fields (name, time, subject)
- Check that your browser isn't blocking notifications

### Calculator not working?
- Try clicking the "Clear" button and starting fresh
- Refresh the page if it seems frozen
